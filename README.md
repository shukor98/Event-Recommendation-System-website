# Event-Recommendation-System-website
This is my first repo in this platform. This project is to recommend event using collaborative filtering. This system is like recommending posting to user based on what post he/she interactive with.

# Installation
To run the system, you must install python, flask, flask-login, Flask-SQLalchemy, numpy, pandas, scipy, sklearn first.

sudo apt install python3-pip
pip3 install flask
pip3 install flask-login
pip3 install Flask-SQLAlchemy
pip3 install numpy
pip3 install pandas
pip3 install scipy
pip3 install sklearn

# Run
To run the system, you just need to change directory to where the project located at and run the command below.

python app.py
